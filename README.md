# mon-package-test

Un package Python installable via pip.

## Installation

```bash
pip install mon-package-test
```

Ou depuis les sources :

```bash
pip install git+https://github.com/henridecargouet-create/mon-package-test.git
```

## Utilisation

```python
from mon_package_test import hello

print(hello("Henri"))  # Hello, Henri!
```

## Développement

```bash
git clone https://github.com/henridecargouet-create/mon-package-test.git
cd mon-package-test
pip install -e .
```
